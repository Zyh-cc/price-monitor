# 国际行情监控

> 实时追踪黄金、石油、白银、比特币等国际大宗商品价格，手机电脑均可流畅使用。

**[📈 立即查看行情 →](https://zyh-cc.github.io/price-monitor/)**

---

## 功能概览

- **滚动行情栏** — 顶部实时滚动展示六大品种最新价格与涨跌幅
- **价格走势卡片** — 黄金 & WTI 原油近一月迷你走势图，一眼看涨跌
- **交互式 K 线图** — 支持切换品种、自定义时间周期，从日内到全历史
- **实时时钟** — 右上角显示当前时间，随时掌握行情时效
- **移动端适配** — 响应式布局，支持 iPhone 刘海屏安全区域

## 使用方式

直接访问 [在线地址](https://zyh-cc.github.io/price-monitor/) 即可，无需注册登录。

- 点击 K 线图上方标签切换品种
- 图表内可自由拖拽缩放时间区间
- 数据由 TradingView 实时提供，延迟极低

## 本地运行

用浏览器直接打开 index.html 即可，无需服务器，无任何依赖。
